# Track UU: KRW Composition

## Overview
Analysis of the Karchmer-Raz-Wigderson (KRW) composition conjecture and its implications.

## KRW Composition Conjecture
For Boolean functions f and g:
CC(f diamond g) = CC(f) + CC(g)

where CC is communication complexity and diamond is function composition.

## Connection to P vs NP
- If KRW holds, then certain circuit lower bounds follow
- This would imply P != NP for non-uniform circuits
- The conjecture is a major open problem in communication complexity

## Key Results

### Theorem UU-1
KRW holds for certain restricted function classes.

### Theorem UU-2
A weak form of KRW implies NC^1 != P.

### Theorem UU-3
The information complexity version of KRW holds.

## CIC Connection
The CIC framework provides:
- An information-theoretic interpretation of KRW
- New tools for attacking the conjecture
- Potential connections to formula size lower bounds
