# P vs NP Research Report: Section 0 - Introduction

## The P vs NP Problem
The P vs NP problem asks whether every problem whose solution can be quickly verified can also be quickly solved. It is one of the seven Millennium Prize Problems.

## Computational Information Complexity
Our approach introduces the Computational Information Complexity (CIC) framework, which measures the information required to describe computational states during algorithm execution.
