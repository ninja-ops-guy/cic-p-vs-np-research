# Useful Programs Specification

## Overview
This document specifies the programs and tools developed during the CIC P vs NP research project.

## SAT Solvers

### cic_sat_solver.py
Basic SAT solver with structural analysis capabilities.

### cic_sat_v2.py
Improved version with portfolio selection.

### cic_sat_v3.py
Advanced version with GNN-based variable ordering.

### cic_sat_v4.py
Production-quality solver with full feature set.

## Analysis Tools

### sdp_treewidth.py
Semidefinite programming approach to treewidth estimation.

### gnn_ordering.py
Graph neural network for variable ordering.

### circuit_ic_framework.py
Circuit complexity analysis framework.

## Benchmark Tools

### fast_benchmark.py
Fast benchmark suite for SAT solvers.

### run_benchmarks.py
Comprehensive benchmark runner.

## MaxSAT Tools

### cic_maxsat.py
MaxSAT solver with width-based optimization.

## Verification

### mcv_verification.py
Minimum Circuit Value problem verification.
